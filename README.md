# Sales_Dashboard

1. 📌 Project Title / Headline

Sales Performance Dashboard | Power BI (Interactive Business Insights)

2. 📖 Short Description / Purpose

This Power BI dashboard provides a comprehensive view of sales performance, profit trends, customer profitability, and product-level insights. It helps stakeholders quickly identify top-performing products, profitable customers, and operational inefficiencies like delivery delays and returns.

3. 🛠️ Tech Stack
Power BI Desktop – Data visualization & dashboard creation
Power Query – Data cleaning & transformation
DAX (Data Analysis Expressions) – KPI calculations & measures
Excel / CSV – Data storage (assumed source format)

4. 📂 Data Source
Retail / Superstore-style dataset (Sales transactions)
Includes fields like:
Order Date
Sales Amount
Profit
Customer Name
Product Name
Category & Segment
Shipping Mode
Delivery Days

5. 🚀 Features / Highlights
📊 Business KPIs
Total Sales: $5.93M+
Total Profit: $747K+
Returned Orders: 2083
Avg. Delivery Days: -9.29 (calculated metric)
🎯 Interactive Filters
Year slicer (2011–2023)
Month filter panel
Segment & Category dropdown filters
📦 Product Insights
Top 10 Products by Sales (Bar Chart)
Identifies best-selling products for revenue optimization
👥 Customer Insights
Top 10 Customers by Profit (Pie Chart)
Helps identify high-value customers
🚚 Operational Insights
Shipping Mode Breakdown (First Class, Same Day, etc.)
Tracks logistics performance
📉 Performance Monitoring
Sales vs Profit tracking
Delivery & return metrics for operational efficiency
